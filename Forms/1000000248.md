# HUD Program Data

* FormID: 1000000248
* Form Type: Edit

## Description

This form collects the basic CSBG required information.

## Form Validation

Questions for youth housing clients will appear if the head of household is under 25.

## Fields

* Required Fields
  * Housing Type
  * Non Cash Benefit
    * Non Cash Benefits List
  * Health Insurance
    * Health Insurance Type(s)
  * Employed
    * Employment Type (if Employed)
    * Why Not Employed (if Not Employed)
  * Marital Status
  * Household Type
  * Disabling Condition
  * Active Military
  * Highest Grade Achieved
* Optional Fields
  * Employer Name
  * Begin Date
  * Pregnant
    * Due Date
  * Youth Housing Questions
* Special Fields
